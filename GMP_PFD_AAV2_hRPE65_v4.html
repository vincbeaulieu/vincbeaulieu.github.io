▼
Culture Parameters

Harvest window:42–48 h post-induction (peak titer); viability 65–75% expected
Staggered Induction Protocol (Novel Feature)
t = 0 h:Doxycycline 1 µg/mL → activates Rep68/40 + adenoviral helpers (GENOME REPLICATION PHASE begins)
t + 8 h:Cumate 30 µg/mL → activates Cap2 expression (CAPSID ASSEMBLY PHASE begins)
Scientific rationale:Temporal decoupling: genomes accumulate BEFORE capsid assembly → 2–4× higher full:empty ratio; 2–4× titer improvement vs simultaneous induction (Lu et al., 2023)
Novelty:PLASMID-FREE production — all AAV components (Rep68/40, Cap2, helper genes, hRPE65 cassette) are stably integrated in the cell genome. Zero GMP plasmids required per production batch.
Expected titer:≥1×10^14 vg/L crude (20 L = 2×10^15 vg total batch input)
vs Luxturna:100–1,000× higher titer than roller-bottle triple-plasmid transfection (10^11–10^12 vg/L)
Novelty: No published GMP or regulatory documentation demonstrates AAV2-RPE65 production in stable inducible HEK293SF lines. This staggered induction approach for this specific vector/disease combination constitutes the core design innovation of this project (High et al., 2025; Jalšic et al., 2023).
